使用数组，我们可以很方便的存储和使用一些数据，所谓数组，从字面也应该可以知道，它是一些数的集合放在一起。

<!-- more -->

**1.数组的创建方法：**

（1）利用 new 创建数组

```js
var arr = new Array()
```

(2)数组字面量创建数组

```js
var arr = [0, 1, 2, 3, 4, 5]
```

注意：数组里面的值中间要用逗号(,)隔开

**2.数组的访问方式**

数组的访问方式是通过数组名＋索引（下标）来进行的

需要特别*注意*的是:

(1)数组的索引是从 0 开始的，依次递增

(2)如果输出时只写了数组名没有写索引，访问的为整个数组

(3)数组的索引号等于数组的长度-1

如：

```js
var arr = [1, 2, 3, 4, 5]
console.log(arr[0]) // 输出结果为 1
console.log(arr[3]) // 输出结果为 4

//数组的长度为5  最大索引为4

console.log(arr) // 输出结果为[1,2,3,4,5]
```

**3.在数组中新增元素**

在数组中新增元素有两种方式：（1）修改 length 的长度 （2）修改索引号，追加数组元素

（1）修改 length 长度

通过修改 length 长度，使数组的长度变大，从而可以存储更多的数据

如:

```js
var arr = [1, 2, 3, 4, 5] // 数组长度为5
arr.length = 7 // 修改数组的长度为7
console.log(arr) // 结果为[1,2,3,4,5,empty,empty]  empty表示为空，未存放数据
arr[5] = 6
console.log(5) // 结果为 6
console.log(6) // 结果为 undefined 未定义
```

(2)修改索引号，追加数组元素

```js
var arr = [1, 2, 3, 4, 5]
arr[5] = 6 // 原来的数组长度为5，通过对数组追加第6个值，使数组的长度增加
arr[2] = 9 // 此时将原来3索引上的值替换
console.log(2) // 结果为9

arr = 10
console.log(arr) // 结果为10 因为对数组进行了赋值，里面的所有值都将被覆盖
```

注意：不要随便给数组赋值，这样会覆盖里面的数据

**3.遍历数组**

遍历数组，即为按顺序将数组中的数据依次打印出来，在遍历数组时，会用到 for 循环语句

如：

```js
        var arr = [1,2,3,4,5];
        for (var i = 0 ； i < arr.length ； i++) {
        console.log(arr[i]);
        };
```

初始化变量 i 必须从 0 开始，因为数组的索引是从 0 开始的。当数组里面的值有很多个时，我们很难直接看出数组的长度，所以在书写判断条件时，写数组的长度 arr.length 更具有普遍性。这里的 i 一次递增对应的为数组的索引号。

总结：利用数组能很好地对数据进行存储和使用
